# DECC 2050 Calculator

Front end for [DECC 2050 calculator](https://github.com/decc/twenty-fifty)

## Development

### Dependencies

* Requires ruby 2.1.x
* Fontcustom has its own dependencies. See
  [github](https://github.com/FontCustom/fontcustom)

### Setup

* `git clone`
* `bundle install`
* `middleman`
